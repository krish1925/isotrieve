Benchmark Methodology

Dataset

  • 300,000 vocabulary items
  • 10,000 test sentences
  • 80/10/10 train/val/test split
  • Zero overlap between sets

Models Tested

  • OpenAI text-embedding-3-small
  • OpenAI text-embedding-3-large
  • Voyage voyage-2
  • Cohere embed-english-v3.0

Metrics

  • Cosine similarity
  • Round-trip fidelity
  • Worst-case analysis
  • Transfer latency

Environment

  • Python 3.12, NumPy 1.26
  • M2 MacBook Pro
  • 10 runs averaged
  • 95% confidence intervals
