Understanding the Results

Semantic Similarity

Measures how well the transferred embedding preserves the original meaning. Higher is better.

  • >90%: Excellent preservation
  • 80-90%: Good quality
  • 75-80%: Acceptable
  • <75%: Poor quality

Information Preserved

Percentage of semantic information retained after round-trip transfer (A→B→A).

AECP typically achieves 86% vs 43% for text-based transfers.

Transfer Latency

Time taken to transfer the embedding between agents.

AECP: <1ms (matrix multiplication)
Text: 150ms+ (re-embedding)

Why AECP is Better

Text serialization loses semantic nuances. AECP preserves the high-dimensional semantic representation directly.

Result: 2x better information preservation with 150x faster transfers.

Try It Yourself

Here's how to implement what you just tested:

from aecp import AECP
from aecp.adapters import OpenAIAdapter, VoyageAdapter

# Create agents
agent_a = AECP(OpenAIAdapter(api_key="sk-..."))
agent_b = AECP(VoyageAdapter(api_key="pa-..."))

# Calibrate (one-time setup)
result = agent_a.calibrate_with(agent_b)
print(f"Calibration quality: {result.validation_similarity:.2%}")

# Transfer embeddings
text = "machine learning and neural networks"
transfer = agent_a.transfer_to(agent_b.agent_id, text)

print(f"Similarity: {transfer.expected_similarity:.2%}")
print(f"Latency: {transfer.latency_ms:.2f}ms")
import { AECP } from '@aecp/core';
import { OpenAIAdapter } from '@aecp/adapters-openai';
import { VoyageAdapter } from '@aecp/adapters-voyage';

// Create agents
const agentA = new AECP({
  embedder: new OpenAIAdapter({ apiKey: 'sk-...' })
});
const agentB = new AECP({
  embedder: new VoyageAdapter({ apiKey: 'pa-...' })
});

// Calibrate (one-time setup)
const result = await agentA.calibrateWith(agentB);
console.log(`Calibration quality: ${result.qualityMetrics.meanSimilarity}`);

// Transfer embeddings
const text = "machine learning and neural networks";
const transfer = await agentA.transferTo(agentB.agentId, text);

console.log(`Similarity: ${transfer.expectedSimilarity}`);
console.log(`Latency: ${transfer.latencyMs}ms`);
# Calibrate agents
curl -X POST https://api.aecp.dev/v1/calibrate \
  -H "Content-Type: application/json" \
  -d '{
    "agent_a": {"model": "text-embedding-3-small", "dimensions": 1536},
    "agent_b": {"model": "voyage-2", "dimensions": 1024},
    "vocabulary_size": 1000
  }'

# Transfer embedding
curl -X POST https://api.aecp.dev/v1/transfer \
  -H "Content-Type: application/json" \
  -d '{
    "from_agent": "agent_a",
    "to_agent": "agent_b",
    "text": "machine learning and neural networks"
  }'
